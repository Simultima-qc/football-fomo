## What this is

**Football FOMO** — "Everything you missed in football today, in 2 minutes."

A daily football digest site. Not a news aggregator, not a social network — a curated daily habit for fans 18-44 who want the essentials without spending an hour across X, Reddit, and YouTube.

Live URL: https://football-fomo.netlify.app

## Automated Daily Pipeline

### How it works

Every day at **6:00 AM UTC**, Netlify runs `netlify/functions/daily-digest-background.mts` automatically (cron job).

```
6:00 UTC
  → Fetch RSS from 8 sources (~52 articles)
  → Deduplicate (skip URLs already in DB today)
  → Analyze up to 40 articles with Claude Haiku (batches of 5)
  → Filter non-football articles
  → Upsert into Supabase `trend_items`
```

### RSS Sources

| Source | Feed |
|---|---|
| BBC Sport | `feeds.bbci.co.uk/sport/football/rss.xml` |
| The Guardian | `theguardian.com/football/rss` |
| Sky Sports | `skysports.com/rss/12040` |
| ESPN FC | `espn.com/espn/rss/soccer/news` |
| L'Équipe | `lequipe.fr/rss/actu_rss_Football.xml` |
| RMC Sport | `rmcsport.bfmtv.com/rss/football/` |
| Transfermarkt | `transfermarkt.com/rss/news` |
| The Athletic | `theathletic.com/rss-feed/` |

### What Claude Haiku produces per article

- Bilingual title (EN + FR)
- 2-sentence summary (EN + FR)
- "Why it matters" (EN + FR)
- Category (one of 9 slugs)
- Scores: `trendScore`, `momentum`, `editorialPriority`, `sourceDiversity`, `eventWeight` (0-100)
- Flags: `mustWatch`, `featured`

### Triggering manually

```bash
curl -X POST https://football-fomo.netlify.app/.netlify/functions/daily-digest-background
```

Response `202` = accepted and running in background. Check logs in Netlify UI → Functions → `daily-digest-background`.

## Database Schema (key tables)

### `trend_items`
| Column | Type | Notes |
|---|---|---|
| `id` | UUID | Generated by function (`crypto.randomUUID()`) |
| `slug` | text | `{analysis-slug}-{YYYY-MM-DD}`, max 80 chars, unique |
| `titleEn` / `titleFr` | text | |
| `shortSummaryEn` / `shortSummaryFr` | text | |
| `whyItMattersEn` / `whyItMattersFr` | text | |
| `sourceUrl` | text | Original article URL |
| `trendScore` | int | Primary sort key |
| `momentum` | int | For "What's Exploding" section |
| `mustWatch` | bool | For Must-Watch section |
| `featured` | bool | |
| `categoryId` | UUID | FK → `categories.id` |
| `publishDate` | timestamptz | Set to today 00:00:00 UTC |
| `updatedAt` | timestamptz | Set at insert time |

### `categories`
9 fixed slugs: `transfers`, `viral-moments`, `matches`, `national-teams`, `club-football`, `world-cup-2026`, `controversies`, `injuries`, `social-buzz`

### `posts`
Manual daily recap posts (type `DAILY_RECAP`, status `PUBLISHED`). Optional — pages work without them.

## Known Issues / Watch Points

- 1 RSS feed returns 404 each run (shown as WARN in logs, non-blocking)
- The Athletic may require auth for full content — summaries may be thin
- Claude Haiku cost: ~$0.01–0.03/day at current volume (40 articles × ~800 tokens)
- `upsert` on conflict `slug` — re-running the same day safely skips duplicates
